# 🏪 Rossmann Store Sales Analysis

## Overview
This project performs exploratory data analysis on Rossmann store sales data to uncover patterns, trends, and key business insights.

The analysis focuses on understanding how sales vary over time, across stores, and under different conditions such as promotions, holidays, and competition.

---

## Problem Statement
Retail businesses need to understand sales behavior to optimize inventory, pricing, and promotions.

This project analyzes historical store sales data to identify trends and factors influencing sales performance.

---

## Dataset Description

The dataset includes:

- Store information
- Daily sales data
- Promotions
- Holidays
- Competition details

---

## Analysis Performed

- Data cleaning and preprocessing  
- Time series analysis (daily, monthly trends)  
- Sales distribution analysis  
- Store-wise performance comparison  
- Impact of promotions on sales  
- Seasonal and trend analysis  

---

## Key Insights

- Sales increase significantly during promotional periods  
- Seasonal patterns affect store performance  
- Certain stores consistently outperform others  
- Holidays influence customer behavior and sales spikes
